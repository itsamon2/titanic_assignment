\# Titanic Survival Prediction



\## Overview

Predict passenger survival using feature engineering and selection.



\## Structure

\- `data/` - Raw datasets

\- `scripts/` - Modular Python scripts

\- `notebooks/` - Jupyter exploration



\## How to Run

1\. Install: `pip install -r requirements.txt`

2\. Run scripts:

&nbsp;  - `python scripts/data\_cleaning.py`

&nbsp;  - `python scripts/feature\_engineering.py`

&nbsp;  - `python scripts/feature\_selection.py`



\## Key Findings

\- Sex is the strongest predictor (18.6% importance)

\- Age and FareLog are next most important

\- Created 5 new features from Name/Fare/Age

\- Final model uses 8 features (reduced from 12)
